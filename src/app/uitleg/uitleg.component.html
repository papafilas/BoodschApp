Waarom Angular?

Dit is eigenlijk geen eenvoudige vraag. Er zijn concurrerende oplossingen, zoals Vue of React, die het ook prima doen.

Iedere variant heeft voordelen, maar kent ook nadelen en toch kan met alle drie een solide frontend worden gebouwd.

Farley verdiep zich in Vue.

Jordy zoekt de React library uit.

En ik onderzoek het Angular framework.

Angular is tegenwoordig een veel gebruikt JavaScript framework. Het wordt gebruikt om dynamische, responsive, zogenaamde Single Page Applications (SPAâ€™s) mee te maken.

Uiteindelijk is een SPA niets anders dan een HTML-pagina, opgemaakt met een vorm van CSS (framework) en uiteindelijk JavaScript. Ik zeg uiteindelijk, omdat in Angular ook gewerkt kan worden met Typescript. Typescript is een superset van Javascript, waarmee je in de ontwikkelomgeving type safe kan werken, maar zodra het naar productie gaat, wordt het gecompileerd naar puur Javascript.
